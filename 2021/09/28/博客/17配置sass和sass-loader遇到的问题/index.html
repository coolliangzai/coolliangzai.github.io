后台node+express+mongodb遇到的问题

  • 跨域问题
  • express中post请求没办法拿到请求体的内容

1.跨域问题的解决

后端配置响应头：
  app.use('/',(req,res,next)=> {
    // 设置请求头为允许跨域
    res.header('Access-Control-Allow-Origin', 'http://localhost:8081');
    // 设置服务器支持的所有头信息字段
    res.header('Access-Control-Allow-Headers', 'Content-Type, Content-Length, Authorization, Accept, X-Requested-With , yourHeaderFeild, sessionToken');
    // 设置服务器支持的所有跨域请求的方法
    res.header('Access-Control-Allow-Methods', 'PUT, POST, GET, DELETE, OPTIONS');
    next()
  },router)

2.express中post请求没办法拿到请求体的内容,已经废弃的用法body-parser

在之前可以通过body-parser来解决
首先安装引入body-parser这个模块
yarn add body-parser 
const app=new express()
const bodyParser=require('body-parser')
// 只要加上这个配置，则在req请求对象上会多出来一个属性：body
// 因此可以直接通过req.body来获取表单post请求体数据
app.use(bodyParser.urlencoded({extended: false}));
app.use(bodyParser.json());

通过express中自带的模块express.json()

const express=require('express')
const app=new express()
//配置express.json()
// 只要加上这个配置，则在req请求对象上会多出来一个属性：body
// 因此可以直接通过req.body来获取表单post请求体数据
app.use(express.json())
require('./plugins/db')()
require('./routes/admin/index.js')(app)
app.listen(8084,()=>{
  console.log('8084端口监听成功');
})

3.后台管理部分配置富文本编辑器

采用的 v-md-editor

安装

# 使用 npm
npm i @kangc/v-md-editor -S

# 使用yarn
yarn add @kangc/v-md-editor

安装支持vue3的版本

# 使用 npm
npm i @kangc/v-md-editor@next -S

# 使用yarn
yarn add @kangc/v-md-editor@next

快速开始

import Vue from 'vue';
import VueMarkdownEditor from '@kangc/v-md-editor';
import '@kangc/v-md-editor/lib/style/base-editor.css';
import vuepressTheme from '@kangc/v-md-editor/lib/theme/vuepress.js';
import '@kangc/v-md-editor/lib/theme/style/vuepress.css';

VueMarkdownEditor.use(vuepressTheme);

Vue.use(VueMarkdownEditor);

在vue3中注册

import { creatApp } from 'vue';
import VueMarkdownEditor from '@kangc/v-md-editor';
import '@kangc/v-md-editor/lib/style/base-editor.css';
import vuepressTheme from '@kangc/v-md-editor/lib/theme/vuepress.js';
import '@kangc/v-md-editor/lib/theme/style/vuepress.css';

VueMarkdownEditor.use(vuepressTheme);

const app = creatApp(/*...*/);

app.use(VueMarkdownEditor);

使用

<template>
  <v-md-editor v-model="text" height="400px"></v-md-editor>
</template>

<script>
export default {
  data() {
    return {
      text: '',
    };
  },
};
</script>
