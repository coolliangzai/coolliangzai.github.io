70.爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

// 首先应找到dp方程，也就是动态规划方程
// 由于爬上n阶台阶的方法数目等于爬上n-1的方法数加上爬上n-2的方法数量的和
const climbStairs=function(n){
  let dp=[]
  dp[0]=1
  dp[1]=1
  if(n===0||n===1)return 1
  for(let i=2;i<n;i++){
    dp[i]=dp[i-1]+dp[i-2]
  }
  return dp[n-1]+dp[n-2]
}

198. 打家劫舍

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下 ，一夜之内能够偷窃到的最高金额。

// dp是一个数组，dp[1]就表示一间房子dp[2]表示两间房子
// dp[1]表示偷一间房间，dp[2]表示偷两间房间
// 由于小偷不能偷盗相邻的房间所以dp[n]=Math.max(dp[n-1],dp[n-2]+nums[n])
var rob = function(nums) {
  // 由于 房间数量不为0,所以至少存在一间房子所以dp[0]=nums[0]
    let dp=[]
    dp[0]=nums[0]
    dp[1]=Math.max(nums[0],nums[1])
    for(let i=2;i<nums.length;i++){
      dp[i]=Math.max(dp[i-1],dp[i-2]+nums[i])s
    }
    return dp[nums.length-1]
};

1025. 除数博弈

爱丽丝和鲍勃一起玩游戏，他们轮流行动。爱丽丝先手开局。
最初，黑板上有一个数字 N 。在每个玩家的回合，玩家需要执行以下操作：
选出任一 x，满足 0 < x < N 且 N % x == 0 。
用 N - x 替换黑板上的数字 N 。
如果玩家无法执行这些操作，就会输掉游戏。
只有在爱丽丝在游戏中取得胜利时才返回 True，否则返回 False。假设两个玩家都以最佳状态参与游戏。

var divisorGame = function(n) {
    let dp=new Array(n+1).fill(false)
    dp[1]=false
    dp[2]=true
    if(n===1)return dp[1]
    if(n===2) return dp[2]
    for(let i=3;i<=n;i++){
        for(let j=1;j<i;j++){
            if(i%j===0&&!dp[i-j]){
                dp[i]=true
                break;
            }
        }
    }
    return dp[n]
};

解题思路：首先我们可以先罗列一下可能的情况：

  1. 当N为1时，爱丽丝不能选也就是题干中的“无法执行这些操作”所以就输掉游戏，返回false。
  2. 当N为2时，由于两个玩家都以最佳状态参与游戏，所以爱丽丝第一次可选的是1，剩余1，鲍勃来操作就只能是输，爱丽丝赢返回true。
  3. 当N为3时，爱丽丝可以选1，因为1能被任何数整除所以可以，N-1为2由前一种条件可知2为true。所以爱丽丝输，返回false。当为2时，2不能被3整除，所以不满足。
  4. 当N为4时，爱丽丝可选1，剩余3，由前一条件可知，当n为3时爱丽丝输，那么4是true
  5. 也就是说dp[i]=N%i===0&&!dp[i-1],dp数组存放的是每一个数字爱丽丝输赢的状态!dp[i-1]表示的是鲍勃输也就是爱丽丝赢

剑指 Offer 42. 连续子数组的最大和

输入一个整型数组，数组中的一个或连续多个整数组成一个子数组。求所有子数组的和的最大值。
要求时间复杂度为O(n)。
示例1:

输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

动态规划最关键的点就是找到状态转移方程以及明确dp数组代表的是什么
本地的dp方程可以表示每一个数字时的最大值
nums = [-2,1,-3,4,-1,2,1,-5,4]
例如当n=0时 dp[0]=nums[0]
当n=1时 由于dp[0]=-2是一个负数，如果和nums[1]相加，反而会让dp[1]的值减小所以dp[1]=nums[1]
当n=2时 由于dp[1]=1是一个正数,所以dp[2]=dp[1]+nums[2]
以此类推dp[i]=dp[i-1]>0?dp[i-1]+nums[i]:dp[i]
最后在求出dp中的最大值
return Math.max(…dp)
代码如下：

var maxSubArray = function(nums) {
    let dp=[]
    dp[0]=nums[0]
    dp[1]=dp[0]>0?dp[0]+nums[1]:nums[1]
    if(nums.length===1) return dp[0]
    if(nums.length===2) return Math.max(...dp)
    for(let i=2;i<nums.length;i++){
        dp[i]=dp[i-1]>0?dp[i-1]+nums[i]:nums[i]
    }
    return Math.max(...dp)
};
